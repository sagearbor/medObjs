Medical School objective DB

A searchable database from multiple national societies



If you want to add objectives to this database you can fill them in using this Medical Objective Upload Excel File Template. Currently updating the database is a manual process, so email the excel to sagearbor@gmail.com to update.

Functionality:

  • Search term - must be present exactly as typed in the objective.
  • Subsearch - after the inital search, the results can be subsearched by entering a second term in box on the results page.
  • Society filter - choose which societies to search (or all)
  • Discipline filter - choose which disciplines to search (or all)
  • Returned columns - choose what data to return
